## Overview
This is the frontend dashboard for Sales Management system built using React and Vite. It provides a clean and responsive interface for searching, filtering, sorting, and paginating large sales datasets in real-time.

## Tech Stack
- React (Vite)
- JavaScript (ES6+)
- CSS

## Features
- Dynamic search with field selection
- Multiple filters (region, gender, age, tags, payemnet method, category, date)
- Sorting controls
- Server-driven pagination
- Real-time table rendering
- Fully responsive layout

## Environment Setup

Create a `.env` file:

```env
VITE_API_BASE_URL=http://localhost:5000/api/sales
```
## Setup Instructions

1. Install dependencies:
```npm install ```

2. Run the development server
``` npm run dev```

3. Open in browser
```http://localhost:5173```
